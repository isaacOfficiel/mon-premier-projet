# mon-premier-projet

## Description

Ce projet est une application en ligne de commande écrite en Python qui permet de gérer une liste de courses.
L’utilisateur peut ajouter, supprimer, afficher ou vider les éléments de la liste à l’aide d’un menu interactif.

Les données sont sauvegardées dans un fichier JSON, ce qui permet de conserver la liste même après la fermeture du programme.
Ce projet m’a permis de pratiquer les bases de Python, la gestion des fichiers et la logique conditionnelle.

## Lancer le projet

Assurez-vous d’avoir Python installé sur votre machine.



(Adapte `liste_de_course`.py au nom réel de ton fichier)

###  Fonctionnalités
## Fonctionnalités

- Ajouter un élément à la liste
- Supprimer un élément
- Afficher la liste
- Vider la liste
- Sauvegarde automatique dans un fichier JSON
